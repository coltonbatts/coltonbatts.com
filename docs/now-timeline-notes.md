# Now Timeline Entries

Add one Markdown file per date in this folder.

- Filename format: `YYYY-MM-DD.md`
- Frontmatter fields:

```yaml
title: Short headline
date: 2026-02-08
summary: One-line summary
links:
  - label: Optional link title
    url: https://example.com
```

Body content is your journal entry for that day. It supports normal Markdown.
